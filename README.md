# Simple Weather Pipeline 🌤️

A simple weather data ETL web application that extracts weather data, processes it using Python and Pandas, and displays the current weather information through a web dashboard.

## Project Overview

This project demonstrates a basic ETL (Extract, Transform, Load) workflow:

- **Extract:** Fetch weather data from a weather source/API
- **Transform:** Clean and process data using Pandas
- **Load:** Display processed weather information on a web dashboard

The main goal of this project is to understand the fundamentals of a data pipeline and how data moves from source to visualization.

## Features

✅ Weather data extraction  
✅ Data processing using Pandas  
✅ Simple ETL workflow  
✅ Web-based weather dashboard  
✅ Displays Mardan weather information  
✅ Python backend with HTML/CSS frontend
